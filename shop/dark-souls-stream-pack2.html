Setup Guide

Alerts

In the alert package folder there is a setup file. Simply click the setup file and follow the instructions.


Stinger Transition
OBS Studio
  1. Unzip the files.
  2. Under the “scene transitions” section, click the "plus" and select “Stinger”
  3. Name the transition to whatever you'd like.
  4. Browse to the "Screens" folder and locate the stinger transition .WEBM file and add it using the browse option in the proporties screen.
  5. Make sure the "Transition Point Type" is set to "Time".
  6. Under "Transition Point (milliseconds)" option, enter "2500". Click OK.
Streamlabs OBS
  1. Unzip the files.
  2. Click the gear icon above your scenes
  3. Set the “Transition” type option to “Stinger”
  4. Set the “Duration” to 5000
  5. Select "Browse" and add the .WEBM file from the "Screens" folder.
  6. Make sure “Transition Point Type” is set to “Time (milliseconds)”
  7. Set the “Transition Point (milliseconds)” to 2500.
  8. Leave the other settings on their defaults and click Done.
Scenes (Starting, BRB, Ending)
  1. Unzip the files and navagate to the "Screens" folder.
  2. If the screen is animated, add it via a "media source". If not add it via a image file.
  3. Set the source to "looping" and be sure to check "Close File When Inactive" option to save CPU.
  4. The blank versions will need to have custom text added via a "text" source.
  5. Inside the screen package folder is a "icon" folder, use the social icons to customize the screen.
  6. The icons are added to the scene via image sources.
  7. Add in text sources, using the recommended font, and place next to the icons if desired.

Overlay Package
  1. Unzip the files.
  2. If the overlay is animated (a video file), add it via a "media source". If not add it via a image file.
  3. Set the source to "looping" (if a video), and be sure to check "Close File When Inactive" option to save CPU.
  4. The blank versions will need to have custom text added via a "text" source.
  5. Inside the overlay package folder is a "icon" folder, use the social icons to customize the overlay if desired.
  6. The icons are added to the scene via image sources.
  7. Add in text sources, using the recommended font, and place next to the icons if desired.
  8. If the overlay is static, the Photoshop file can also be used to customize the overlay with text and icons.
